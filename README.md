## 🔒 Fonctionnalités de Sécurité

Pour protéger les ressources pédagogiques:

1. **Filigrane sur les Documents** - IAFECOLE - Documents Protégés
2. **Désactivation du Clic Droit** - Impossibilité de copier les contenus
3. **Blocage de la Sélection** - Interdiction de sélectionner du texte
4. **Raccourcis Bloqués** - F12, Ctrl+U, Ctrl+Shift+I désactivés
5. **Avertissements** - Messages clairs sur les restrictions

> **Note:** Les captures d'écran et enregistrements d'écran ne peuvent pas être totalement bloqués par un navigateur. Les protections web possibles ont été implémentées avec des messages clairs.

## 🚀 Guide d'Utilisation

### Pour les Étudiants:

1. **Accédez à la Plateforme**
   - Visitez IAFECOLE
   - Explorez la page d'accueil

2. **Recherchez des Ressources**
   - Utilisez la barre de recherche
   - Filtrez par matière, type de document, ou session
   - Naviguez par le menu latéral

3. **Utilisez le Tuteur IA** 🤖
   - Cliquez sur le bouton "Tuteur IA"
   - Posez vos questions
   - Recevez des explications instantanées

4. **Générateur de Quiz** ✏️
   - Cliquez sur "Quiz"
   - Sélectionnez une matière et un niveau
   - Testez vos connaissances

5. **Réviseur de Cours** 📖
   - Cliquez sur "Réviseur"
   - Collez vos notes
   - Recevez une analyse et des recommandations

6. **Téléchargez les Documents**
   - Prévisualisez avant de télécharger
   - Cliquez sur "Télécharger"
   - Exploitez les ressources
